# 새롭게 알게된 점
  - docker container를 새로 만들지 않고 이전에 만들어둔 container를 사용하는 것도 된다. 
    - docker start [name]
    - detached mode 가 가능하다. `docker run -d`
    - attached mode 도 가능하다. 
    - docker start 는 detach mode가 default, docker run 은 attach mode 가 default
